# BTEC RAG Assistant

This repository contains a Streamlit-based Retrieval-Augmented Generation (RAG) assistant for BTEC learning materials. The app uses local approved documents, ChromaDB embeddings, and Groq LLM integration to answer user questions from validated BTEC sources.

## Key features

- Reads approved local files only (PDF, DOCX, TXT)
- Fetches an approved Pearson BTEC PDF from an external URL
- Creates embeddings using `sentence-transformers`
- Stores vectors in a local `ChromaDB` database
- Uses semantic retrieval to answer queries from approved sources only
- Provides a Streamlit UI for generating text, PDF, or PowerPoint outputs
- Includes simple source attribution and similarity filtering

## How it works

1. The app scans the `data` folder for supported files and loads them.
2. It extracts text from PDFs, DOCX, and TXT documents.
3. It chunks the text, creates embeddings, and stores them in `chroma_db`.
4. User questions are embedded and used to retrieve the closest approved text chunks.
5. The answer is generated by a Groq-backed LLM using only the retrieved context.
